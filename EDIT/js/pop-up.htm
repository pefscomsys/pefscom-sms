Options
To edit the size of the popup, open the "pop-up.js" in Notepad or a plain text editor. Edit "var popup_width" and "var popup_height". You can also turn on any option by replacing the "0" with a "1". Example: change "toolbar=0" to "toolbar=1" to have a toolbar in the pop-up.

Edit the "pop-upstyle.css" that comes in the .zip file to change the colors for this text and the page title.

You can even add graphics and other links to this page. Edit this page like any other HTML page.
